# 🇮🇳 India Growth & Development Dashboard

A collection of Python-based economic analytics projects focused on analyzing India's economic history, crisis periods, reforms, and development journey using data analysis, statistics, and visualization.

This repository combines multiple interconnected projects related to:

- Economic Crisis Analysis
- Economic Reforms
- GDP Growth
- Inflation
- Forex Reserves
- Economic Indicators
- Historical Economic Events

The goal of this repository is to build a long-term economic analytics project series using Python and Data Science.

# 📦 Installation

Clone the repository:

```bash
git clone https://github.com/25f2005869-glitch/india-growth-dashboard.git
```

Go to repository folder:

```bash
cd india-growth-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run any project:

```bash
python analysis.py
```

or

```bash
python dashboard.py
```
